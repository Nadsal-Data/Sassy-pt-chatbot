# 💅 Fatbot: The Sarcastic AI Personal Trainer 

> **"Bestie, you give your toxic ex more chances than your diet, and it’s giving delulu."**

Welcome to **Fatbot**, the only personal trainer that roasts you into shape using TikTok lingo and zero patience for your excuses. Built with **Streamlit** and **Azure OpenAI**, this bot is designed to keep it 100 while helping you get those gains.

---

## ✨ Features

* **AI-Initiated Roasts:** The bot starts the convo. You don't even have to say hi; it’s already judging your lifestyle.
* **Excuse Guardrails:** Try saying "I'm tired" or "I'll start Monday." The bot has built-in triggers to shut down your weak energy before it even wastes your API tokens.
* **TikTok Lingo Integration:** Responses are flavored with "no cap," "periodt," "caught in 4k," "biih," and "queen."
* **Multi-Turn Memory:** It remembers your previous messages using a sliding window memory.
* **GIF Support:** Every roast is paired with a sarcastic GIF because a picture is worth a thousand calories you didn't burn.

## 🛠️ Setup Instructions

1. **Clone the repo:** `git clone https://github.com/your-username/fatbot.git`
2. **Install dependencies:** `pip install -r requirements.txt`
3. **Run the App:** `streamlit run streamlit_app.py`
